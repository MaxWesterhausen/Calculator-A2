#Taschenrechner 

Der Taschenrechner besteht aus einem Programm in Visual Studio welches das Menü für den Rechner definiert. 
Dieses Programm soll mit einem zweiten Programm kommunizieren, welches auf einem Arduino läuft. 
Auf dem Arduino soll dann die eigentliche Rechnung stattfinden. 
Die Kommunikation zwischen PC und Microcontroller läuft über eine serielle Schnittstelle also den USB-Port (COM3) des PCs. 
Dazu Müssen Windows die richtigen Befehle gegeben werden, damit es die Kommunikation zulässt.
